# fk-zdnj

これは [雑談Slack Advent Calendar 2022](https://adventar.org/calendars/8098) の2日目のやつです。

## 用例

### hello.zdnj

```
ざざざざざざざざざだ雑ざざざざざざざざざざざ談つん雑ざざざざざzdnjつつつzdnjざざざざざざざzdnjzdnjざざざzdnj
```

#### 実行

```sh
$ ./zatsudan hello.zdnj
hello
```
